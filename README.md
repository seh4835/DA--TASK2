# Student Grade Calculator

This project is a simple Python program that takes a student's name and marks, then assigns a grade with an encouraging message.

## Features
- Uses if-elif-else for grading
- Validates input (0–100 only)
- Uses while loop for invalid inputs
- Uses functions
- Displays encouraging messages

## Grading Logic

| Marks Range | Grade |
|-------------|-------|
| 90–100      | A     |
| 80–89       | B     |
| 70–79       | C     |
| 60–69       | D     |
| 0–59        | F     |
